# Decision Tree Classification Project

## Project Overview
This project demonstrates the implementation of the Decision Tree Classification algorithm using Python and Streamlit.

The model predicts class labels based on input features using a tree-based learning approach.

---

## Machine Learning Type
- Supervised Learning
- Classification
- Decision Tree Classifier

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

---

## Dataset
The project uses a classification dataset for prediction.

Example:
- Iris Dataset
- Student Pass Prediction
- Customer Churn Dataset

---

## Features Used
Example features:
- Age
- Salary
- Study Hours
- Experience

---

## Target Variable
The target variable contains class labels such as:
- Yes / No
- Pass / Fail
- 1 / 0
